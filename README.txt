GitHub Upload Instructions (Static Site)
=======================================

Files included:
- index.html
- styles.css
- script.js
- assets/images/* (your attached photos)

How to upload to GitHub:
1) Open your GitHub repository.
2) Click "Add file" > "Upload files".
3) Drag and drop ALL files/folders from this zip.
4) Commit changes.
5) If using GitHub Pages, ensure the site is published from the correct branch/folder.

Notes:
- You can rename index.html later (e.g., non-oem-touch-ideck.html) if you want this page under your Parts site.
- If this page will live at a custom domain path, update the canonical URL in index.html.
- The "Open Parts Verification Guide" button currently links to:
  https://tovis-aus.github.io/Parts/
